
## なぜこれを作ったか

20代後半、月給14万からスタートした。技術がないわけじゃなかった。要領も悪くない。それでも、年収はなかなか動かなかった。運用保守からSES、SIerと環境を変えても、同じ景色が続く感覚は変わらなかった。

そこで気づいたのは、付加価値は「何ができるか」じゃなく、「お金がどこに動いているか」と接続して初めて値段がつく、ということだった。AD、Azure、運用設計。誰もが避けるニッチでも、お金が流れてくる側に身体を置けば、未経験からでも単価は跳ねる。技術だけ磨いても年収は上がらない。お金がどこに動いていて、自分の何を乗せれば届くのか。インフレが進む時代に、そこに目を向けないのは本当にもったいないと思う。  

もちろん、技術は妥協しない。探求心を持って手を動かし続ける、それが前提の上での話。お金の軸はその上に乗っかってくるだけだ。  

そして、楽したいから努力しないんじゃなくて、楽したいから努力する。最低限の力で這い上がる方法を、ずっと探してきた。気づけば、毎年100万ずつ年収が上がり、借金300万を返し終え、資格は30個になっていた。Xには化け物みたいな人がいて、自分は本当に最小限しかやっていない。だから「才能でしょ」で済ませず、再現性のある話として渡せる。

このLPは、商品販売ではない。LINE登録と無料相談予約への補助ページ。同じ運用保守やSESで燻ってる人、AIに乗りたいけど何から触ればいいか分からない人、フリーランスに興味はあるけど勇気が出ない人 — そういう人と繋がりたくて作った。説教したいわけでも、煽りたいわけでもない。ちょっと話して、キャリアの霧が少し晴れて、次の3年で何をすべきかの輪郭が見えれば、それで十分だと思っている。

ラフでいきましょう。
